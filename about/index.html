Tentang aku

   halo nama saya Norman Andrianyah. biasa dipanggil Norman. membuat dan memprogram HTML, CSS, javascript, C, C ++, java, GLSL, PBR dan POM. selamat datang di blog pertama saya. Saya tinggal di negara kelahiran Cilacap dan tinggal di Cimahi, saya berusia 13 tahun. keahlian saya bermain video game dan coding html, css, glsl. Saya juga memiliki perusahaan kecil bernama Arthenal Graphics ;P

    saya membuat peta Normal dan peta Specular untuk Minecraft. Saya belajar paket resourcepack untuk Minecraft sejak 2018. Awalnya saya belajar untuk edisi saku, Sejak saya melihat Youtube, saya ingin membuat paket tekstur yang realistis. tetapi saya tidak berpikir itu mungkin. setelah itu saya melihat di youtube, dan saya belajar dari Grafik Continuum

   pertama setelah saya mengenal game Minecraft, saya penasaran dan ingin memainkannya, tetapi sayangnya game itu dibayar. jadi saya memutuskan untuk mencari game yang mirip Minecraft, lalu saya menemukannya. ternyata yang saya dapatkan adalah versi lama dari Minecraft. ternyata juga lebih buruk dari yang saya bayangkan, saya hanya punya beberapa blok dan lebih sedikit item. bahkan selama petualangan hanya ada beberapa area di dunia. dan setelah itu saya memutuskan untuk menghapusnya. Ngomong-ngomong, itu adalah kisah saya ketika saya berusia 10 tahun dan itu juga mencari di Google

   Saya mencari gim yang menurut saya memenuhi kejenuhan, saya melihat iklan di YouTube yang gimnya mirip dengan Minecraft. permainan ini disebut MWBA (Mini World: Block Art). ada game seperti PvP, daripada dugaan saya bahwa Minecraft tidak dapat melakukan Multiplayer. Saya dulu masih kecil, jadi saya tidak tahu apa-apa tentang Minecraft. setahun kemudian saya mengunduh Minecraft beta di 5play.ru versi 1.9, di situlah saya mengenal Minecraft. Saya bebas melakukan apa pun dengan perintah. Saya tidak pernah berpikir itu lebih baik daripada MWBA

cerita saya sebelum mengenal Minecraft dan membuat tekstur

   beberapa minggu saya melihat shader yang membuat Minecraft Realistis, tetapi setelah diinstal paket source tidak berfungsi, karena versi gl dan variabel berbeda. lihat betapa bodohnya saya bahwa ada video khusus untuk Minecraft PE, tapi itu hanya thumbnail dari gambar orang lain menggunakan edisi java. dan tidak cocok untuk menggambar atau bisa dikatakan video itu hanya clickbait, jadi itu hanya tipuan, saya mencari MCPE, tapi itu juga sama, hanya thumbnail dan clickbait.

   beberapa bulan kemudian, saya mulai mencoba membuat Resourcepack saya sendiri tanpa melihat tutorial di YouTube, yang mana cukup keren? Saya membuat paket sumber daya yang awalnya merupakan tekstur bajakan, selain itu saya sedikit mengeditnya dan juga yang terburuk diunggah ke MCPE Master. ilegal kan? Saya memutuskan untuk menghapusnya dan mencoba untuk tidak membuat resourcepack lagi. Apalagi shadernya juga bajakan.

    beberapa tahun kemudian saya memiliki PC (Personal Computer), saya menggunakannya sebagai coding, bermain game, dll. beberapa bulan kemudian saya mencari minecraft edisi java gratis. namanya TLauncher, saya dulu berpikir TLauncher sangat mirip dengan Minecraft Launcher pada umumnya. tapi saya sadar setelah saya mencoba membuat Realm dan juga memainkan server premium seperti Hypixel. tapi saya bisa bersyukur untuk itu karena saya dapat menginstal shader, begitu juga fitur peta Normal dan peta Specular. jadi saya bisa menikmatinya

    saat itulah saya mulai mencoba membuat resourcepack untuk Minecraft Java edition, di resourcepack ada fitur seperti Normal, dan juga peta specular. itu membuat Minecraft saya lebih mudah dimainkan. beberapa minggu kemudian saya mencoba membuat paket sumber daya yang mirip dengan Ray Tracing yang disebut Arthenal. Saya mencoba membuat tekstur karena saya ingin melanjutkan generasi Ray Tracing Resourcepack oleh anak-anak Indonesia. jadi di situlah Arthenal dilahirkan untuk membuat Minecraft lebih realistis dengan PBR dan POM / Normals dan Specular.
