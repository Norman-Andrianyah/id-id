About

   halo nama saya Norman Andrianyah. biasa dipanggil Norman. membuat dan memprogram HTML, CSS, GLSL, PBR dan POM. selamat datang di blog pertama saya. Saya tinggal di negara kelahiran Cilacap dan tinggal di Cimahi, saya berusia 13 tahun. keahlian saya bermain video game dan coding html, css, glsl. Saya juga memiliki perusahaan kecil bernama Arthenal Graphics ;P

   saya membuat peta Normal dan peta Specular untuk Minecraft. Saya belajar paket resourcepack untuk Minecraft sejak 2018. Awalnya saya belajar untuk edisi saku, Sejak saya melihat Youtube, saya ingin membuat paket tekstur yang realistis. tetapi saya tidak berpikir itu mungkin. setelah itu saya melihat di youtube, dan saya belajar dari Continuum Graphics

   pertama setelah saya mengenal game Minecraft, saya penasaran dan ingin memainkannya, tetapi sayangnya game itu dibayar. jadi saya memutuskan untuk mencari game yang mirip Minecraft, lalu saya menemukannya. ternyata yang saya dapatkan adalah versi lama dari Minecraft. ternyata juga lebih buruk dari yang saya bayangkan, saya hanya punya beberapa blok dan lebih sedikit item. bahkan selama petualangan hanya ada beberapa area di dunia. dan setelah itu saya memutuskan untuk menghapusnya. Ngomong-ngomong, itu adalah kisah saya ketika saya berusia 10 tahun dan itu juga mencari di Google

   Saya mencari gim yang menurut saya memenuhi kejenuhan, saya melihat iklan di YouTube yang gimnya mirip dengan Minecraft. permainan ini disebut MWBA (Mini World: Block Art). ada permainan seperti PvP, daripada dugaan saya bahwa Minecraft tidak dapat melakukan Multiplayer. Saya dulu masih kecil, jadi saya tidak tahu apa-apa tentang Minecraft. setahun kemudian saya mengunduh Minecraft beta di 5play.ru versi 1.9, di situlah saya mengenal Minecraft. Saya bebas melakukan apa pun dengan perintah. Saya tidak pernah berpikir itu lebih baik daripada MWBA

   beberapa minggu saya melihat shader yang membuat Minecraft Realistis, tetapi setelah diinstal paket source tidak berfungsi, karena versi gl dan variabel berbeda. lihat betapa bodohnya saya bahwa ada video khusus untuk Minecraft PE, tapi itu hanya thumbnail dari gambar orang lain menggunakan edisi java. dan tidak cocok untuk menggambar atau Anda bisa mengatakan bahwa video itu hanya clickbait, jadi itu hanya tipuan, saya sedang mencari MCPE, tapi itu juga sama, hanya thumbnail dan clickbait.

   beberapa bulan kemudian, saya mulai mencoba membuat Resourcepack saya sendiri tanpa melihat tutorial di YouTube, yang mana cukup keren? Saya membuat paket sumber daya yang awalnya merupakan tekstur bajakan, selain itu saya sedikit mengeditnya dan juga yang terburuk diunggah ke MCPE Master. ilegal kan? Saya memutuskan untuk menghapusnya dan mencoba untuk tidak membuat paket sumber daya lagi. Apalagi shadernya juga bajakan.




    beberapa tahun kemudian aku mempunyai PC (Personal Computer), aku menggunakan itu sebagai coding, main games, dll. beberapa bulan kemudian aku mencari minecraft java edition yang gratis. namanya TLauncher, dulu aku anggap TLauncher sudah sangat mirip seperti Minecraft Launcher pada umumnya. tapi aku menyadarinya setelah aku mencoba membuat Realm dan juga bermain diserver premium seperti Hypixel. tapi aku bisa mensyukuri itu karena bisa memasang shader, begitu juga ada fitur Normal map dan Specular map. jadi aku bisa menikmatinya

    disaat itulah. aku mulai mencoba membuat resourcepack untuk Minecraft Java edition, di resourcepack tersebut ada fitur seperti Normal, dan juga specular map. itu membuat Minecraftku lebih enak untuk dimainkan. beberapa minggu kemudian aku mencoba untuk membuat resourcepack mirip Ray Tracing yang disebut Arthenal. aku mencoba membuat texture tersebut karena aku ingin meneruskan generasi Ray Tracing Resourcepack karya anak bangsa Indonesia. jadi disitulah, Arthenal lahir untuk membuat Minecraft lebih realsitis dengan PBR dan POM/Normals dan Speculars.
