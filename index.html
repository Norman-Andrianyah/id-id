TexturePacks

    Paket tekstur atau disebut juga resourcepack untuk Minecraft, adalah proyek kami. kami membuat paket texturepack / resourcepack untuk Minecraft untuk mengubah tekstur blok, item, dll. dan mengapa saya membuatnya? banyak pemain Minecraft adalah pengembang tekstur, karena untuk menggantinya sangat mudah. Selain itu, ada juga pengembang Shader untuk Minecraft. Kami membuat berbagai tekstur, tetapi dengan peta normal dan specular, sehingga teksturnya lebih realistis dari biasanya. Tekstur saya juga dibuat sendiri, jika Anda ingin berpartisipasi dalam membuat tekstur, hubungkan GitHub kami. Tekstur POM dan PBR bukan tekstur biasa, peta normal dan peta spekular, hanya dapat dibuat pada perangkat lunak tertentu
